# Mixed-Type Data Anomaly Detection System 🔍

This is the official repository for my Final Year Project. The system integrates **LightGBM** for detection and **SHAP** for interpretability to handle mixed-type data (numerical + categorical) effectively.

## 🚀 Key Features
- **Hybrid Model:** Handles mixed data types natively without complex preprocessing.
- **Explainable AI:** Provides global and local explanations using SHAP values.
- **Interactive UI:** Built with Streamlit for real-time anomaly simulation.
